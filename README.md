# SProject&nbsp;[![License: MIT](https://img.shields.io/badge/license-MIT-blue.svg)](https://opensource.org/licenses/MIT) ![C++](https://img.shields.io/badge/Language-C++-00599C?style=flat&logo=cplusplus&logoColor=white) ![UnrealEngine](https://img.shields.io/badge/Engine-UnrealEngine-0E1128?style=flat&logo=unrealengine&logoColor=white) ![UnrealEngineVersion](https://img.shields.io/badge/Version-5.5.6-0E1128?style=flat&logo=unrealengine&logoColor=white)

`SProject`는 언리얼 엔진 기반의 개인 프로젝트로, 다양한 기능과 시스템을 포함하고 있습니다.

## 요구 사항
 - Unreal Engine 5.5 이상
 - C++17 이상 (Visual Studio 2022 권장)
 - 플러그인
    - [GGameCore](https://github.com/dek0058/GGameCore)
    - SProjectAssets - 게임 에셋(비공개 레포지토리)
